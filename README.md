# 🌌 SF Popup Generator

코코포리아(CCFOLIA) 마커 패널용 SF 컷인 이미지를 생성하는 웹 애플리케이션입니다.
업로드한 이미지를 기반으로 점(Dot)에서 시작해 화려하게 전개되는 홀로그램 애니메이션(WebP)을 자동 생성합니다.

## ✨ 기능 소개
- **SF 애니메이션 자동 생성**: 단일 이미지를 WebP 포맷의 애니메이션으로 변환합니다.
- **부드러운 이펙트**: 렌즈 플레어 응축 및 홀로그램 전개 효과 적용.
- **컬러 커스터마이징**: `app.py` 내부의 색상 변수 단 한 줄로 UI와 이펙트 색상을 일괄 변경할 수 있습니다.

## 🚀 실행 방법
이 프로젝트는 Python 및 Gradio를 사용하여 제작되었습니다.

1. 저장소를 클론하거나 다운로드합니다.
